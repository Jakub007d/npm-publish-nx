# @jakub007d/math-utils

Simple math utility functions for addition, subtraction, multiplication, and division.

## Installation

```bash
npm install @jakub007d/math-utils
```

## Usage

```typescript
import { add, subtract, multiply, divide } from '@jakub007d/math-utils';

add(2, 3);        // 5
subtract(10, 4);  // 6
multiply(3, 4);   // 12
divide(20, 5);    // 4
```
