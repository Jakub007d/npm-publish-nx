# @jakub007d/string-utils

Simple string utility functions for common string operations.

## Installation

```bash
npm install @jakub007d/string-utils
```

## Usage

```typescript
import { toUpperCase, toLowerCase, capitalize, reverse, isPalindrome } from '@jakub007d/string-utils';

toUpperCase('hello');       // 'HELLO'
toLowerCase('WORLD');       // 'world'
capitalize('hello world');  // 'Hello world'
reverse('hello');           // 'olleh'
isPalindrome('racecar');    // true
```
